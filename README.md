# Portfolio Website

**Overview**

This project is a personal portfolio website .

The website demonstrates the use of HTML, CSS, and Tailwind CSS to create a responsive and mobile-first design. It contains multiple pages that showcase personal information, projects, skills, education, and contact details.

**Features**

- Responsive mobile-first layout
- Navigation between multiple pages
- Resume section with education, skills, and experience
- Project showcase pages
- Contact form with validation
- Tailwind CSS styling

**Technologies Used**

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Git & GitHub

**Pages**

- Home
- About
- Resume
- Projects
- Project Details
- Contact
